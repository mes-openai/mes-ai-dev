# Governance Rules Index

> 本文件用于导航治理类规则，包括审查、门禁、阶段产物、共享知识写入与骨架治理等。
> 治理类规则仅在需要约束交付形式、审查方式或共享写入时加载。

---

## 一、适用场景

以下情况应优先查看治理规则：

- 需要输出正式审查报告。
- 需要确认阶段产物布局与命名。
- 需要进行 Completion Sweep。
- 需要处理共享知识写入。
- 需要修改骨架规则、模板或标准产物。
