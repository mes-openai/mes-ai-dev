# 发布说明模板

> 本模板用于发布交付阶段，记录版本变更信息和升级指南。
> 本模板**保持独立**，不并入 `delivery-package-template.md`：
> - `delivery-package-template.md` 负责交付范围、审计、验收与签收
> - 本模板负责版本对外说明、功能/API/数据库变更摘要、升级与回滚指南
> 在交付场景中，建议两者配套使用：交付母版记录“是否可交付”，发布说明模板记录“本版本具体变了什么、如何升级”。
> 使用说明：将 `[请填写xxx]` 替换为实际内容。
