# 交付包母版

> 本母版统一承载交付范围说明、交付包审计、验收结果三类常见交付场景。
> 使用方式：按实际场景选择重点章节填写；不适用章节可标注“本次不适用”。
> 若仅需快速选型：
> - 交付边界说明 → 重点填写“交付范围”“未完成项”“已知限制”
> - 交付包审计 → 重点填写“产物核对”“包与配置审计”“回滚验证”
> - 验收报告 → 重点填写“验收标准检查”“测试覆盖验收”“验收结论”
> - 版本对外说明 / 升级指南 → 使用独立的 `release-note-template.md`

## 七、使用规则

1. 所有交付类文档优先使用本母版，不再优先新建近似模板
2. 交付范围说明、交付审计、验收结论可共用同一母版，不适用章节标注“不适用”即可
3. 若涉及正式发布，回滚方案与综合结论必须填写
4. 若需要保留历史文件名，旧模板文件可作为入口页引用本母版，不再维护独立章节副本
