# 阶段完成产物报告模板

> 本模板用于在每个阶段完成后，统一说明本阶段标准产物、实际已生成产物、未生成产物及原因。
> 属于阶段退出门禁的强制产物，未生成或内容不达标不得进入下一阶段。
> 若与通用详细审查报告模板存在重叠，仍必须满足 `.opencode/references/mes-ai-reference/templates/governance/detailed-review-report-template.md` 的最小字段集要求。
