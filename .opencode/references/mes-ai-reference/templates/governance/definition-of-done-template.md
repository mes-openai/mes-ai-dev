# 完成定义模板

> 本模板用于定义阶段级或工作单元级 DoD（Definition of Done）。
> DoD 用于回答“什么叫完整完成”、“什么叫 GSD 完成”、“什么叫不完成”。
> 所有 GSD Continue Exit 都必须能够映射到某个明确的 GSD 完成定义。

## 六、结论规则

- 满足完整完成定义 → 标记完整完成
- 不满足完整完成但满足 GSD 完成 → 标记 GSD 完成
- 两者都不满足 → 标记不完成
