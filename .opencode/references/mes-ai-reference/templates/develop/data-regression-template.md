## 处理规则

1. **MES必做**：涉及工单、库存、追溯、OEE/SPC的变更，数据回归验证为必做项
2. **基线快照先行**：验证前必须先建立基线数据快照
3. **计算精度**：涉及金额/数量的计算，精度必须与业务规则一致（如：保留4位小数）
4. **100%通过率**：数据回归校验要求100%通过，不允许部分通过
