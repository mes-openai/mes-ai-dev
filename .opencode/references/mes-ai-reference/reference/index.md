# Reference 总索引

> 本文件用于集中导航项目参考资料、知识消费入口、结构说明与规范索引。
> 本文件只做导航，不展开详细正文。使用时按当前任务范围按需加载。

---

## 一、使用原则

- 仅加载与当前任务直接相关的参考文档。
- 优先读取摘要性、索引型文档，再按需进入详细正文。
- 不得因为存在索引而默认全量加载全部 Reference 文档。

---

## 二、推荐加载顺序

1. 先看本索引，确认需要哪一类参考信息。
2. 再进入对应专题索引或单文档。
3. 必要时再进入详细规则、模板或源码。
