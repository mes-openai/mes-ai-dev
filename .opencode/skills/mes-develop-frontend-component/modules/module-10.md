# 审核要点

## 审核要点

### 公共组件复用
- [ ] 是否优先使用公共组件
- [ ] 是否有重复开发公共组件
- [ ] 公共组件使用是否正确
- [ ] 公共组件 props 传递是否正确

### 组件职责清晰
- [ ] 组件职责是否单一
- [ ] 是否有过度复杂的组件
- [ ] 是否有合理的组件拆分
- [ ] 是否有清晰的组件层级

### Props 定义正确
- [ ] 所有 props 是否有 type 定义
- [ ] 所有 props 是否有注释
- [ ] 默认值是否合理
- [ ] required 标记是否正确

### 事件定义正确
- [ ] 事件命名是否规范
- [ ] 事件参数是否完整
- [ ] emit 调用是否正确
- [ ] 父组件是否正确监听

### 样式规范
- [ ] 是否使用 scoped
- [ ] 类名是否使用 BEM
- [ ] 样式是否与现有风格一致
- [ ] 是否有样式变量使用

### 模板规范
- [ ] HTML 结构是否清晰
- [ ] 指令使用是否正确
- [ ] 组件引用是否正确
- [ ] 是否有注释说明

### 功能正确性
- [ ] 数据加载是否正确
- [ ] 搜索过滤是否正确
- [ ] 分页处理是否正确
- [ ] 操作处理是否正确

### 代码质量
- [ ] 是否有冗余代码
- [ ] 是否有重复逻辑
- [ ] 是否有潜在 bug
- [ ] 是否有性能问题
