# 输出模式说明

## 输出模式说明

本 Skill 支持两种输出模式：

### 模式 A：基础索引模式
适用于 `/mes-init-project` 的基础建图阶段。

必需产物：
- `index.md`

最小要求：
- 完成 Schema 概览
- 完成风险画像
- 完成关键对象摘要
- 若未识别热表/共享表/大表/回滚敏感对象，必须明确写“本次未识别”

可选产物：
- `tables.md`
- `relations.md`

### 模式 B：深化明细模式
适用于 `/mes-init-enrich` 或明确声明需要表结构/关系深化的场景。

必需产物：
- `index.md`
- `tables.md`（若深化目标包含表结构明细）
- `relations.md`（若深化目标包含关系分析）

最小要求：
- `index.md` 在基础画像上补强风险说明
- `tables.md` 包含实际表结构摘要
- `relations.md` 包含实际关系分析，或明确说明“未识别关系”
