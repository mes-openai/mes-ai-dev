# graph-consumers-not-deploy-scope 资源说明

## 图谱导读

- GitNexus 提示本次变更接口存在多个下游消费者。
- 该导读只用于发现候选影响面、回归检查点和部署风险提示。

## 正式证据

- 真实发布包清单。
- 目标环境与发布窗口。
- 依赖确认记录。
- 回滚条件、回滚对象与人工签核材料。

## 冲突处理

当图谱导读与真实发布包、环境、依赖或回滚条件冲突时，以真实发布证据和交付阶段门禁材料为准。图谱结果只能作为待确认风险，不得替代部署计划结论。

## 依赖边界

本 eval 仅模拟 GitNexus 导读文本，不引入真实 GitNexus / graphify 运行依赖，也不修改任何真实 deploy / release 脚本。
