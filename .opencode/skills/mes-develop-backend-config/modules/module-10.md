# 环境变量清单

## 环境变量清单

| 变量名 | 默认值 | 说明 | 是否必须 |
|--------|--------|------|----------|
| DB_HOST | localhost | 数据库主机 | 是 |
| DB_PORT | 3306 | 数据库端口 | 是 |
| DB_USER | root | 数据库用户 | 是 |
| DB_PASSWORD | password | 数据库密码 | 是 |
| OTHER_SERVICE_HOST | localhost | 其他服务主机 | 是 |
| OTHER_SERVICE_PORT | 8080 | 其他服务端口 | 是 |
