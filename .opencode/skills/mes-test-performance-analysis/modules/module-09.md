# 审核要点

## 审核要点

### N+1查询
- [ ] 是否避免循环中的单条查询
- [ ] 是否使用批量查询方法
- [ ] 嵌套对象是否优化加载

### 大事务
- [ ] 事务范围是否合理
- [ ] 是否避免远程调用在事务中
- [ ] 是否避免嵌套事务

### 慢SQL
- [ ] 是否使用索引字段查询
- [ ] 是否避免全表扫描
- [ ] 是否合理使用JOIN

### 索引
- [ ] 查询条件字段是否有索引
- [ ] 排序字段是否有索引
- [ ] 组合索引是否合理

### 数据量
- [ ] 是否合理分页
- [ ] 是否避免全量返回
- [ ] 批量操作是否分批
