### 6. 编写DML脚本
创建文件：`mes-ai-dev/workspace/development/{REQ-ID}/database/dml-xxx.sql`

**DML脚本结构**：
```sql
-- ============================================================================
-- 需求编号：REQ-YYYYMMDD-XXX
-- 功能描述：数据初始化/迁移
-- 作者：AI-Assistant
-- 创建时间：YYYY-MM-DD HH:mm:ss
-- ============================================================================

-- ----------------------------------------------------------------------------
-- 数据初始化：XXX表
-- ----------------------------------------------------------------------------
INSERT INTO `schema_name`.`table_name`
  (`column1`, `column2`, `created_by`, `created_time`)
VALUES
  ('value1', 'value2', 'system', NOW()),
  ('value3', 'value4', 'system', NOW());

-- ----------------------------------------------------------------------------
-- 数据迁移：XXX表迁移到YYY表
-- ----------------------------------------------------------------------------
INSERT INTO `schema_name`.`new_table`
  (`column1`, `column2`, `created_by`, `created_time`)
SELECT
  `old_column1`,
  `old_column2`,
  'migration',
  NOW()
FROM `schema_name`.`old_table`
WHERE `status` = 'ACTIVE';
```

### 7. 编写回滚脚本
创建文件：`mes-ai-dev/workspace/development/{REQ-ID}/database/rollback-xxx.sql`

**回滚脚本结构**：
```sql
-- ============================================================================
-- 需求编号：REQ-YYYYMMDD-XXX
-- 功能描述：回滚脚本
-- 作者：AI-Assistant
-- 创建时间：YYYY-MM-DD HH:mm:ss
-- ============================================================================

-- ----------------------------------------------------------------------------
-- 回滚：删除表
-- ----------------------------------------------------------------------------
DROP TABLE IF EXISTS `schema_name`.`table_name`;

-- ----------------------------------------------------------------------------
-- 回滚：删除索引
-- ----------------------------------------------------------------------------
ALTER TABLE `schema_name`.`table_name`
  DROP INDEX `idx_column_name`;

-- ----------------------------------------------------------------------------
-- 回滚：删除数据
-- ----------------------------------------------------------------------------
DELETE FROM `schema_name`.`table_name`
WHERE `created_by` = 'migration';
```

**Step Gate B**：DDL/DML/回滚脚本存在命名、约束、安全性或可回滚性缺口 → 打回步骤4-7重做，不得交付迁移方案设计。
