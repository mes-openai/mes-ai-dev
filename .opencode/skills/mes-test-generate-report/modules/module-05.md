# 输入依赖

## 输入依赖

| 输入项 | 路径 | 说明 |
|--------|------|------|
| test-cases | `mes-ai-dev/workspace/testing/REQ-YYYYMMDD-XXX/test-cases.md` | 测试用例文档 |
| integration-tests | `mes-ai-dev/workspace/testing/REQ-YYYYMMDD-XXX/integration-tests.md` | 集成测试文档 |
| 单元测试结果 | `jalor/服务名/target/surefire-reports/` | JUnit测试报告 |
| coverage-results | 覆盖率统计输出或覆盖率报告 | 本轮纳入验证范围代码的覆盖率证据 |
| test-execution-evidence | 测试执行结果/门禁记录/验证证据 | 用于确认新生成测试用例是否全部通过、是否存在删测 |
| test-report-template | `mes-ai-dev/templates/test-report-template.md` | 测试报告模板 |
| spec | `mes-ai-dev/workspace/requirements/REQ-YYYYMMDD-XXX/spec.md` | OpenSpec 需求规格主文档 |
