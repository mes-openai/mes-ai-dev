# 审核要点

## 审核要点

### 字段正确性
- [ ] Entity字段是否与数据库表字段一一对应
- [ ] 字段类型是否正确映射数据库类型
- [ ] 主键生成策略是否正确
- [ ] 外键关联是否正确配置

### 注解完整性
- [ ] Entity是否有@Entity、@Table注解
- [ ] 字段是否有@Column注解（如需要）
- [ ] DTO是否有校验注解
- [ ] VO是否有格式化注解

### 命名一致性
- [ ] 类名是否符合命名规范
- [ ] 字段名是否符合命名规范
- [ ] 是否与现有代码风格一致

### 注释完整性
- [ ] 是否有类注释
- [ ] 所有字段是否有注释
- [ ] 注释是否清晰准确

### 代码质量
- [ ] 是否实现Serializable接口
- [ ] 是否有serialVersionUID
- [ ] 是否使用Lombok简化代码
- [ ] 转换逻辑是否正确

### 类型安全
- [ ] 日期类型是否统一
- [ ] 数字类型是否匹配
- [ ] 字符串长度是否一致
- [ ] 空值处理是否合理
