# 包结构

## 包结构

\`\`\`
[服务目录]/
├── src/main/java/com/xxx/
│   ├── controller/    # API端点
│   ├── service/       # 业务逻辑
│   ├── dao/           # 数据访问
│   ├── entity/        # 数据实体
│   ├── dto/           # 数据传输对象
│   ├── vo/            # 视图对象
│   ├── config/        # 配置类
│   └── util/          # 工具类
└── src/main/resources/
    ├── application.yml
    └── restService.properties
\`\`\`
