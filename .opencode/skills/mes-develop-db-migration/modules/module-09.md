# 审核要点

## 审核要点

### 迁移顺序
- [ ] 顺序是否合理（依赖关系）
- [ ] 是否先加后删
- [ ] 是否先数据后结构

### 回滚方案
- [ ] 是否每个操作都有回滚
- [ ] 回滚脚本是否完整
- [ ] 回滚是否可执行

### 数据校验
- [ ] 是否有数据量校验
- [ ] 是否有关键字段校验
- [ ] 是否有关联关系校验

### 风险评估
- [ ] 是否评估锁表风险
- [ ] 是否评估迁移时间
- [ ] 是否评估服务影响
- [ ] 是否有缓解措施

### 前置检查
- [ ] 是否检查磁盘空间
- [ ] 是否检查备份状态
- [ ] 是否检查表锁状态

### 执行建议
- [ ] 是否建议执行时间
- [ ] 是否说明是否需停机
- [ ] 是否有应急预案
