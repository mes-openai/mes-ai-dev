# 数据库迁移方案

## 数据库迁移方案

### 需求编号
REQ-YYYYMMDD-XXX

### 变更范围
- 新增表：X个
- 新增字段：X个
- 新增索引：X个
- 数据迁移：X条
- 删除字段：X个
- 删除表：X个

### 迁移顺序

| 步骤 | 操作类型 | SQL语句 | 预估耗时 | 是否可回滚 |
|------|----------|---------|----------|------------|
| 1    | 创建表   | CREATE TABLE xxx | 1秒 | 是 |
| 2    | 新增字段 | ALTER TABLE ADD COLUMN | 10秒 | 是 |
| 3    | 创建索引 | CREATE INDEX | 60秒 | 是 |
| 4    | 数据迁移 | INSERT INTO ... | 300秒 | 是 |
| 5    | 删除字段 | ALTER TABLE DROP COLUMN | 5秒 | 是（有回滚脚本） |

### 前置检查

| 检查项 | 检查SQL | 期望结果 |
|--------|---------|----------|
| 磁盘空间 | SHOW TABLE STATUS | 剩余空间 > 表大小 × 2 |
| 表锁检查 | SHOW OPEN TABLES | 无未释放的锁 |
| 备份状态 | 检查备份日志 | 最新备份时间 < 24小时 |

### 回滚方案

| 操作 | 回滚SQL | 回滚脚本文件 |
|------|---------|--------------|
| CREATE TABLE xxx | DROP TABLE xxx | rollback-xxx.sql |
| ALTER TABLE ADD COLUMN | ALTER TABLE DROP COLUMN | rollback-xxx.sql |
| 数据迁移 | DELETE FROM ... | rollback-xxx.sql |

### 数据校验方案

| 校验项 | 校验SQL | 校验时机 |
|--------|---------|----------|
| 数据量 | SELECT COUNT(*) | 迁移前后 |
| 关键字段 | 比对关键字段 | 迁移后 |
| 关联关系 | 检查孤儿数据 | 迁移后 |

### 风险评估

| 风险项 | 风险等级 | 影响范围 | 缓解措施 |
|--------|----------|----------|----------|
| 大表DDL锁表 | 中 | 服务写入受影响 | 在低峰期执行，设置等待超时 |
| 数据迁移耗时 | 高 | 数据不完整窗口期 | 分批迁移，提供中间状态查询 |
| 回滚复杂度 | 低 | 回滚需恢复数据 | 保留备份表，记录原数据 |

### 执行时间建议
- 建议执行时间：凌晨2:00-4:00（低峰期）
- 预估总耗时：X分钟
- 是否需要停机：否/是（如需停机说明原因）

### 执行步骤
1. 备份数据库
2. 执行前置检查
3. 执行DDL脚本（按顺序）
4. 执行DML脚本（如有）
5. 执行数据校验
6. 确认无误后清理临时数据
7. 如有问题，执行回滚脚本

### 应急预案
- 迁移失败处理：立即执行回滚脚本
- 数据不一致处理：停止服务，恢复备份
- 服务不可用处理：通知用户，提供降级方案
```

**Step Gate B**：迁移顺序、回滚方案、数据校验或风险评估未闭环 → 打回步骤4-8重做，不得交付部署计划阶段使用。
