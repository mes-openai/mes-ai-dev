# 审核要点

## 审核要点

### SQL正确性
- [ ] SQL语法是否正确
- [ ] 表名、字段名是否与数据库一致
- [ ] 参数映射是否正确
- [ ] 结果映射是否完整

### 功能完整性
- [ ] 是否包含必要的CRUD操作
- [ ] 是否支持批量操作
- [ ] 是否支持条件查询
- [ ] 是否支持分页查询

### 性能优化
- [ ] 是否避免全表扫描
- [ ] 是否使用索引字段
- [ ] 批量操作是否合理
- [ ] 分页查询是否高效

### 代码规范
- [ ] 命名是否规范
- [ ] 格式是否统一
- [ ] 注释是否完整
- [ ] XML格式是否正确

### 风格一致性
- [ ] 是否与现有Mapper风格一致
- [ ] SQL片段命名是否一致
- [ ] ResultMap定义是否一致
- [ ] 动态SQL使用是否一致
- [ ] `namespace` 是否与 Mapper 接口全限定名一致
- [ ] XML `id` 是否与接口方法一一对应

### 安全性
- [ ] 是否使用参数化查询
- [ ] 是否防止SQL注入
- [ ] 删除操作是否有条件限制
- [ ] 敏感数据是否有权限控制
