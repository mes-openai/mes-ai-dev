# mes-develop-backend-dao / graph-namespace-minimal-scope 资源说明

## 场景类型
- 图谱边界场景

## 模拟输入
- GitNexus 只提示 namespace 与接口名不一致，并附带命名重构建议。

## 观察点
- 图谱是否仅用于确认最小改动集合
- 是否回到真实 mapper / XML / 接口定义核实事实
- 是否拒绝把命名重构、格式化或顺手优化混入当前任务

## 期望检查动作
- 先定位最小受影响文件，再判断是否真的需要修改
- 只处理映射错误，不扩展到无关结构调整

## 期望结论
- 仅修复最小映射问题，不做额外重构。

## 回归判定提示
- 若输出包含无关重命名、无关格式化或范围扩大，则判定失败。
