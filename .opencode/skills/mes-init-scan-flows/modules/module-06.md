# 约束规则

## 约束规则

### 必须遵守
1. **执行前输出计划** — 在任何操作前必须输出执行计划
2. **上下文管理** — 调用 `mes-guard-context-budget` Skill 评估上下文消耗
3. **层级限制** — 优先读第0层和第1层，只在推断需要时读第2层源码
4. **文件大小限制** — 遵守 `mes-ai-dev/knowledge/rules/artifact-standards.md` 的文档大小限制

### 禁止行为
1. ❌ 不修改已有知识库文件
2. ❌ 不读取前端代码（本 Skill 只关注后端业务链路）
3. ❌ 不猜测不确定的链路（标注为"待确认"）
