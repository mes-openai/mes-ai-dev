# 审核要点

## 审核要点

### SQL注入
- [ ] 是否使用参数化查询
- [ ] 是否避免字符串拼接SQL
- [ ] 动态SQL是否安全

### XSS
- [ ] 输入是否校验/转义
- [ ] 输出是否编码
- [ ] 富文本是否过滤

### 越权访问
- [ ] API是否有权限校验
- [ ] 数据操作是否校验归属
- [ ] 批量操作是否安全

### 数据泄露
- [ ] 日志是否打印敏感信息
- [ ] 响应是否包含敏感字段
- [ ] 错误信息是否安全

### 接口鉴权
- [ ] 是否有认证校验
- [ ] 是否有防刷机制
- [ ] 公开接口是否标注

### 文件上传
- [ ] 文件类型是否限制
- [ ] 文件大小是否限制
- [ ] 存储路径是否安全

### CSRF
- [ ] 重要操作是否有CSRF防护
- [ ] 是否有二次确认机制
