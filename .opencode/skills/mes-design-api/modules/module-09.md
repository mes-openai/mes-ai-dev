**基本信息**
| 项目 | 内容 |
|------|------|
| 服务归属 | [微服务名称] |
| Controller | [类名] |

**请求参数**
| 参数名 | 位置 | 类型 | 必填 | 说明 |

| ... | ... | ... | ... | ... |

```json
{
  "name": "示例名称",
```

| 字段名 | 类型 | 说明 |
|--------|------|------|
| code | Integer | 响应码 |

| data | Object | 响应数据 |
| ... | ... | ... |

**成功响应示例**
```json
{
  "code": 200,
  "message": "操作成功",
  "data": {
    "id": 1,
    "name": "示例"
  }
}
```

**错误响应示例**
```json
{
  "code": 40001,
  "message": "参数校验失败"

```
