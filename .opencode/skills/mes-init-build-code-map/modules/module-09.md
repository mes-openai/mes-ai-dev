# 使用指南

## 使用指南

### 快速定位

1. **查找某个服务的详细信息**
   - 第0层：backend-overview.md → 找到服务名
   - 第1层-精简版：services/service-xxx/index.md → 快速确认相关性
   - 第1层-完整版：services/service-xxx/detail.md → 查看详情

2. **查找某个文件位置**
   - 第1.5层：services/service-xxx/file-summaries.md 或 modules/module-xxx/file-summaries.md → 精准定位文件

3. **查找某个API**
   - 第2层：api-registry.md → 搜索路径

4. **查找数据库表结构**
   - 第2层：database-registry.md → 找到Schema归属
   - 第3层：schema-xxx/tables.md → 查看表结构

5. **查找前端功能对应的后端**
   - 第0层：frontend-overview.md → 找到模块
   - 第1层-精简版：modules/module-xxx/index.md → 快速确认模块
   - 第1层-完整版：modules/module-xxx/detail.md → 查看详情
   - 第2层：frontend-backend-map.md → 查看映射
