## 处理规则

1. **三阶段分离**：环境冒烟 → 业务冒烟 → 灰度观察，严格按顺序执行
2. **环境冒烟不通过 = 不进入业务冒烟**
3. **业务冒烟不通过 = 不进入灰度发布**
4. **灰度回滚 = 必须记录回滚原因和处理结果**
5. **观察指标最少8项**：接口成功率 + 错误码 + 响应时间 + 核心链路 + 关键表增量 + MQ + 缓存 + 锁
