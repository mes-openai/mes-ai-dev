# 阶段门禁使用标准摘要

## 一、门禁类型

- 阶段进入门禁
- 步骤消费门禁
- 阶段退出门禁
- 汇总消费门禁

---

## 二、强制触发时机

1. analyze / design / develop / test / deliver / emergency / refresh 进入前
2. 每个步骤输出被下游消费前
3. 任一阶段退出前
4. 共享知识收口、数据库迁移、高风险发布等关键场景前后

---

## 三、最小内容

门禁报告至少包含：

- 门禁类型
- 检查目标
- 检查对象
- 检查依据
- 检查项与结果
- 问题清单
- 风险评估
- 门禁结论
- 证据路径
- 后续动作
