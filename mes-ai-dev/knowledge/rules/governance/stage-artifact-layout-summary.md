# 阶段产物目录标准摘要

## 一、标准目录

每个阶段 `REQ-YYYYMMDD-XXX` 目录下默认包含：

- `deliverable/`
- `report/`
- `evidence/`
- `handoff/`
- `memory/`
- `working/`

---

## 二、固定文件位置

- `report/stage-output-report.md`
- `memory/pitfall-ledger.md`
- `memory/decision-log.md`
- `memory/blocker-log.md`

---

## 三、阶段最小产物

- 全部标准目录
- `stage-output-report.md`
- 至少一份详细审查报告
- 符合 OpenSpec 的阶段主交接文档
- 三类 memory 台账
